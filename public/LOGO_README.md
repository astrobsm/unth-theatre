# Logo Placement Instructions

## Adding Your Logo

1. Place your logo file in the `public/` directory with one of these names:
   - `logo.png` (recommended)
   - `logo.svg` (best for scalability)
   - `logo.jpg`

2. For best results, your logo should be:
   - Transparent background (PNG or SVG)
   - Minimum 200px width
   - High resolution for clarity

The application will automatically use your logo in:
- Login page
- Registration page
- Dashboard sidebar
- Navigation header
- All pages
